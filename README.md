# TRATA – Women's Safety Device

## Problem
Women often face unsafe situations where immediate help is not available.

## Solution
A wearable device that triggers an alert when a button is pressed or abnormal force is detected. The system activates a buzzer and is designed to send location-based alerts using GSM and GPS.

## Components
- Arduino Mega
- Button module
- Buzzer and LED
- Force sensor
- Piezo sensor
- GPS
- GSM 
- Battery
- Charging module
